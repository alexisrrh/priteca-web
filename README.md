# PRITECA – Web de Reformas y Construcción

Sitio web moderno desarrollado para una empresa de reformas, construcción y obras en España.  
Diseñado para transmitir profesionalidad, captar clientes y mostrar trabajos realizados de forma visual.

---

## 🚀 Tecnologías utilizadas

- React (Vite)
- JavaScript (ES6+)
- Tailwind CSS
- React Router DOM
- Framer Motion (animaciones)

---

## 🏗️ Funcionalidades principales

- Página de inicio con diseño moderno y visual
- Sección de servicios:
  - Reformas de pisos
  - Reformas de chalets
  - Obras públicas
  - Piscinas
- Galería de obras tipo scroll horizontal (estilo moderno)
- Comparación Antes / Después
- Página de servicios detallada
- Página de obras con galería visual
- Página de contacto con formulario
- Botón flotante de WhatsApp
- Animaciones suaves al hacer scroll
- Diseño totalmente responsive (móvil y escritorio)

## ⚙️ Instalación y uso

Clona el repositorio:

```bash
git clone https://github.com/alexisrrh/priteca-web.git
cd priteca-web
Instala dependencias:

npm install

Ejecuta en desarrollo:

npm run dev
